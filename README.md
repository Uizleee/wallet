## Installation

Из корня проекта запустить:
1. npm install
2. docker-compose up --build 
3. npm run migration:run
4. npm run start:dev
5. по адресу http://localhost:3000/graphql - смотреть доступные методы

## Running the app

```bash
# development
$ npm run start

# watch mode
$ npm run start:dev

# production mode
$ npm run start:prod
```
